# Obesity Predictor

A complete end to end  ML workflow for an ``Obesity Predictor`` using `Regression` and `Classification`

1. Performed Exploratory Data Analysis
2. Performed `BMI prediction` using Regression.
3. Predicted `Obesity level` using Classification models.


## About Data

This dataset include data for the estimation of obesity levels in individuals from the countries of Mexico, Peru and Colombia, based on their eating habits and physical condition.

The Dataset has been collected from the official UC Irvine ML repository.

`https://archive.ics.uci.edu/dataset/544/estimation+of+obesity+levels+based+on+eating+habits+and+physical+condition`
## Deployment

To deploy this project run

```bash
  python data/EDA.ipynb 
  python data/REGRESSION_MODEL.ipynb
  python data/CLASSIFICATION_MODEL.ipynb
```
